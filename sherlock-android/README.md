# Sherlock Android

OSINT mobile application built with Kotlin + Jetpack Compose.

## Build

This app is built automatically via GitHub Actions on every push to `sherlock-android/**`.
The debug APK artifact is available for download from the Actions tab.

## Features

130+ OSINT features including:
- Username search across 400+ platforms
- Email breach checking (HIBP)
- Domain/IP/Phone lookup
- Hash identification & lookup
- Pastebin dump search
- Wayback Machine integration
- Biometric app lock
- End-to-end encryption
- Incognito mode
- And much more...
