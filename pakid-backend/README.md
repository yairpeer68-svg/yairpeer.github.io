# פקיד — הפעלת השרת 🏛️

האפליקציה ב-`/pakid/` עובדת כרגע במצב הדגמה. המדריך הזה מסביר איך מדליקים את השרת האמיתי — משתמשים, מסד נתונים ומנוע ה-AI. **זמן משוער: 20 דקות. עלות: ₪0 להתחלה** (חוץ מקרדיט API קטן ל-AI).

## שלב 1: יצירת פרויקט Supabase (חינם)

1. היכנס ל-https://supabase.com ולחץ **Start your project** (הרשמה עם חשבון GitHub — הכי מהיר)
2. **New project** ← שם: `pakid` ← בחר סיסמה למסד הנתונים (שמור אותה) ← Region: אירופה (eu-central) ← **Create**
3. המתן ~2 דקות שהפרויקט יקום

## שלב 2: הקמת מסד הנתונים

1. בתפריט הצד: **SQL Editor** ← **New query**
2. העתק את כל התוכן של הקובץ `schema.sql` (בתיקייה הזאת) ← הדבק ← **Run**
3. אמור להופיע "Success" — זהו, יש לך מסד נתונים עם משתמשים, מכסות וניתוחים

## שלב 3: הפעלת כניסה אנונימית

1. **Authentication** ← **Sign In / Up** (או Providers)
2. הפעל **Anonymous Sign-Ins** ← Save
   (ככה משתמשים מתחילים בלי שום טופס הרשמה — חיכוך אפס)

## שלב 4: מפתח AI

1. היכנס ל-https://console.anthropic.com ← הרשמה
2. **Billing** ← טען קרדיט התחלתי של $5 (מספיק למאות ניתוחים; כל ניתוח עולה ~2-4 סנט)
3. **API Keys** ← **Create Key** ← העתק את המפתח (מתחיל ב-`sk-ant-`)

## שלב 5: פריסת פונקציית הניתוח

1. ב-Supabase: **Edge Functions** ← **Deploy a new function** ← בחר "Via Editor"
2. שם הפונקציה: `analyze-letter`
3. מחק את קוד הדוגמה, הדבק את כל התוכן של `functions/analyze-letter/index.ts` ← **Deploy**
4. **Edge Functions ← Secrets** ← הוסף:
   - Name: `ANTHROPIC_API_KEY`, Value: המפתח משלב 4

## שלב 6: חיבור האפליקציה

1. ב-Supabase: **Project Settings ← API** והעתק שני ערכים:
   - **Project URL** (למשל `https://abcd1234.supabase.co`)
   - **anon public key**
2. פתח את הקובץ `pakid/index.html` במאגר, מצא למעלה את:
   ```js
   const CONFIG = {
     SUPABASE_URL: "",
     SUPABASE_ANON_KEY: "",
   };
   ```
   הדבק את שני הערכים בין המרכאות, שמור ודחוף ל-main (או בקש מקלוד לעשות את זה — פשוט שלח לו את שני הערכים).
3. אחרי שהאתר מתעדכן: התג למעלה יתחלף מ"גרסת הדגמה" ל"מחובר לשרת ●", וצילום מכתב אמיתי יעבור ניתוח AI מלא.

## מה קיבלת

- **משתמשים אמיתיים** — כל מכשיר מקבל חשבון אוטומטית (אנונימי), מוכן לשדרוג להרשמה מלאה
- **מכסה מובנית** — 3 ניתוחים בחינם בחודש לכל משתמש; פרימיום = בלי הגבלה (עמודת `is_premium` בטבלת profiles)
- **היסטוריה בענן** — כל הניתוחים נשמרים בחשבון המשתמש
- **אבטחה** — Row Level Security: כל משתמש רואה רק את הנתונים של עצמו; מפתח ה-AI חי רק בשרת ולעולם לא נחשף ללקוח

## עלויות בקנה מידה

| משתמשים פעילים | Supabase | AI (בהנחת 3 ניתוחים/משתמש/חודש) |
|---|---|---|
| עד ~1,000 | ₪0 (תוכנית חינמית) | ~$90/חודש — ולכן המכסה החינמית קטנה והפרימיום בתשלום |
| 10,000+ | ~$25/חודש | מכוסה מהכנסות הפרימיום |

המודל העסקי מובנה: המשתמש החינמי טועם (3 ניתוחים), ומי שצריך יותר — משלם ₪19.90/חודש שמכסים בענק את עלות ה-AI שלו.
